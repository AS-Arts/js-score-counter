# js-score-counter
Simply add up the points of your game for each player. Enter the number of players and get started!

[demo & more details](https://theartsblog.de/spielergebnisse-zusammenzaehlen-mit-jquery-projekt-scorecount/)


## Features

* Bootstrap 5.3
* Bootstrap Icons
* jQuery 3.6.3
* Up to 12 players
* Simple code
* Full Responsive
* Automatically adds new rounds
* Calculates the average
* Player names can be changed
* No data is saved
